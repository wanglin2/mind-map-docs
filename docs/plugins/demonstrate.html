DemonstrateжҸ’д»¶жҸҗдҫӣжј”зӨәеҠҹиғҪгҖӮ

иҝӣе…Ҙжј”зӨәжЁЎејҸж—¶дјҡиҮӘеҠЁе°Ҷе®№еҷЁе…ғзҙ е…ЁеұҸпјҢз„¶еҗҺй»ҳи®ӨиҒҡз„ҰеҲ°ж №иҠӮзӮ№пјҢеҸҜйҖҡиҝҮй”®зӣҳж–№еҗ‘й”®зҡ„е·ҰеҸіжқҘеҲҮжҚўдёҠдёҖжӯҘе’ҢдёӢдёҖжӯҘпјҢеҸҜйҖҡиҝҮEscй”®йҖҖеҮәжј”зӨәжЁЎејҸгҖӮ

иҝӣе…Ҙжј”зӨәжЁЎејҸеҗҺжҖқз»ҙеҜјеӣҫжүҖжңүзҡ„еҝ«жҚ·й”®йғҪе°Ҷж— жі•дҪҝз”ЁпјҢйј ж Үд№ҹж— жі•ж“ҚдҪңжҖқз»ҙеҜјеӣҫгҖӮ

й…ҚзҪ®иҜ·еҸӮиҖғе®һдҫӢеҢ–йҖүйЎ№гҖӮ

import Demonstrate from 'simple-mind-map/src/plugins/Demonstrate.js'
