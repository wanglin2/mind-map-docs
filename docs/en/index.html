SimpleMindMap

A powerful web mind map

It is a JS mind mapping library and a complete mind mapping tool

Online use
Quick start

Rich functionality

It has both the functions of mind maps that you can think of and those that you can't imagine.

Pluggable

Except for the core functions, all other capabilities are provided as plugins for on-demand selection, reducing unnecessary code.

Easy to use

Ready to use out of the box, easy to get started, rich in configuration, and clear documentation.

It's both a JS library and a tool

Not only is it a framework independent JS library, but it also provides a fully functional web version and client-side mind mapping tool.
